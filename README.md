# TrainerOS

TrainerOS is a monorepo containing the web frontend, backend API, admin app, and mobile app for the TrainerOS platform.

## Projects

- `frontend` - React + Vite marketing site and app shell
- `backend` - Express + Prisma API
- `admin` - admin interface
- `mobile` - mobile client

## Local development

Run each app from its own folder:

```bash
cd backend && npm run dev
cd frontend && npm run dev
```
